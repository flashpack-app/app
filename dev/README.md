# Development Scripts

This folder contains utility scripts for common development tasks.

## Usage

### Mac/Linux
Make scripts executable:
```bash
chmod +x dev/*.sh
```

Run scripts:
```bash
./dev/git-pull.sh
./dev/start.sh
./dev/clear-cache.sh
```

### Windows
Run scripts:
```cmd
dev\git-pull.bat
dev\start.bat
dev\clear-cache.bat
```

## Available Scripts

- **git-pull.sh/bat** - Pull latest changes from git
- **start.sh/bat** - Start Expo development server
- **clear-cache.sh/bat** - Clear Expo cache and restart
